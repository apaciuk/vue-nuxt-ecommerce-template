## Installation

Make firebase db and put details in plugins/firebase.js

$ npm install  (dont use yarn)

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
